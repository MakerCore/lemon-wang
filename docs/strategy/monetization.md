# lemon.wang 变现方案 v2

> 品牌定位：SOLO. SOUR. SOVEREIGN.  
> 核心能力：AI硬件PM + Maker实战 + PM决策框架 + Writer表达  
> 更新日期：2026-04-26  
> 版本：v2（聚焦订阅优先，执行顺序严格）

---

## 执行原则（核心）

1. **一个一个来，不要都一起搞**
2. **先跑通订阅，再扩展其他**
3. 变现顺序：订阅 → 内容电商 → 工具/SaaS → 咨询/内训 → 联盟/广告

---

## 一、订阅层设计（第一优先级）

订阅是当前唯一需要全力推进的变现模式。其他模式在订阅跑通之前不启动。

### 1.1 订阅价值主张

**核心问题：订阅者为什么需要 lemon.wang？**

| 用户类型 | 核心问题 | lemon.wang提供的价值 |
|---------|---------|---------------------|
| 海外用户（Marcus/Elena） | 为什么中国团队看起来不创新，但却屡次做出爆品？ | 出海视角：如何在低成本基础上构建产品力 |
| 国内用户（硬件PM） | 技术出身但不知道怎么产品化、变现 | 欧美的商业思维、市场思维、用户思维 |

**订阅提供的具体价值：**
- 每周深度分析：AI硬件产品拆解、供应链洞察、Maker实战复盘
- 决策框架：PM方法论、硬件产品判断逻辑
- 社区归属：同频独立创作者/Maker网络

### 1.2 Substack（海外订阅）

**定位：** 面向欧美市场，提供"中国产品视角"的深度分析

**订阅价值：**
- 每周1篇深度英文文章（产品拆解、市场分析、Maker访谈）
- 月度"硬件趋势追踪"报告
- 季度Q&A（订阅者专属）
- Indie maker社区入口

**定价：**
- $9/月 或 $79/年（约$6.6/月，年付省25%）
- 首月体验：$1（获取初始订阅者）

**免费→付费转化路径：**
```
X threads（免费洞察）
    ↓
完整分析在Substack（付费墙后）
    ↓
订阅者加入（$9/月）
```

**具体操作：**
1. X上每周发布3-5条threads，内容是文章精华/思考片段
2. 文章正文放在Substack付费墙后，threads末尾引导"Full analysis on Substack"
3. 目标：X followers → Substack订阅，转化率目标5-10%

### 1.3 爱发电（国内订阅）

**定位：** 面向国内Maker/硬件PM，提供一手实战复盘

**与Substack的差异：**
| 维度 | Substack（海外） | 爱发电（国内） |
|-----|-----------------|---------------|
| 内容 | 英文、面向欧美读者、中国产品视角 | 中文、面向国内读者、实战复盘 |
| 深度 | 偏分析、框架、方法论 | 偏实操、避坑、Maker记录 |
| 社区 | 英文indie maker社区 | 国内Maker/PM圈子 |

**订阅价值：**
- 每周专属newsletter（国内视角）
- 爱发电专属帖子（随想、数据、工具推荐）
- 问答优先回复
- 早期训练营/课程优先参与权

**定价：**
- ¥49/月 或 ¥399/年（约¥33/月，年付省33%）

**免费→付费转化路径：**
```
公众号免费文章
    ↓
底部"相关阅读" → 爱发电订阅
    ↓
或：免费PDF/决策框架卡片留下邮箱
    ↓
邮件序列引导订阅爱发电
```

**具体操作：**
1. 公众号每篇文章末尾加"订阅爱发电"CTA
2. 免费PDF作为引流钩子，填邮箱获取，邮件序列引导订阅
3. 目标：公众号粉丝 → 爱发电订阅，转化率目标3-5%

### 1.4 订阅启动指标

| 指标 | 3个月目标 | 6个月目标 | 12个月目标 |
|-----|---------|---------|---------|
| Substack订阅者 | 30 | 100 | 300 |
| 爱发电订阅者 | 50 | 150 | 500 |
| 公众号粉丝 | 3,000 | 10,000 | 30,000 |
| X followers | 2,000 | 5,000 | 15,000 |

**订阅跑通标准（月经常性收入）：**
- 跑通基准：MRR达到 ¥5,000+$500/月
- 扩展基准：MRR达到 ¥15,000+$1,500/月

## 二、内容 → 电商 路径（订阅跑通后第二步）

### 2.1 电商选品方向

**前提：** 订阅跑通后（约6-12个月后）再考虑电商

**可能产品方向：**
1. **决策框架工具包**：Notion/Airtable模板 → AI硬件PM决策框架实现
2. **Maker工具包**：硬件项目常用元器件/模块包
3. **信息产品**：PDF合集、案例库、检查清单

**与内容的协同：**
- 内容验证需求 → 电商产品化
- 例如：内容中反复提到的"硬件决策框架" → 做成Notion模板销售

### 2.2 电商启动条件

- 爱发电订阅≥200
- 已验证的内容主题经过市场验证
- 有一定忠实粉丝群体

## 五、联合创始人/投资人咨询机会

### 5.1 内容影响力 → 这类机会的路径

**核心逻辑：**
- 持续输出AI硬件/出海/Maker相关内容
- 建立"专业洞见者"形象
- 被动吸引机会，而非主动推销

**具体路径：**

1. **Substack文章 → 投资人发现**
   - 深度分析AI硬件投资机会的文章
   - 被投资人订阅 → 主动联系

2. **X活跃度 → 创业圈曝光**
   - 持续输出关于硬件产品、供应链的见解
   - 被想创业的创始人关注 → 潜在co-founder机会

3. **Maker内容 → 创业团队需要**
   - 记录自己的Maker项目过程
   - 有硬件背景的创始人主动接触 → 顾问/co-founder角色

### 5.2 加速这类机会的行动

- **每月1篇深度投资分析**（Substack）：建立"懂硬件投资"形象
- **每周X互动**：

在硬件创始人聚集的社区活跃（Indie Hackers、Product Hunt评论区）

- **Maker项目公开化**：记录自己的硬件项目过程，展示"能做"的能力

### 5.3 注意事项

- 这类机会是"被选中"，不是"主动追求"
- 核心还是先把订阅和内容影响力做起来
- 保持开放但不过度分心

## 六、执行节奏（严格按顺序）

### Phase 1（第1-6个月）：订阅跑通

**核心目标：** Substack和爱发电订阅达到初步规模

**每月任务：**
- 公众号：每周2篇
- X：每周10-15条threads
- Substack：每周1篇深度英文文章
- 爱发电：每周1篇专属中文内容

**KPI检查点：**
- 第3个月：Substack 20人，爱发电 30人
- 第6个月：Substack 100人，爱发电 150人

### Phase 2（第7-12个月）：订阅稳定 + 电商/咨询小规模测试

**订阅稳定化：**
- 维持内容质量，专注提升续费率
- 建立社区互动

**小规模测试：**
- 电商：推出第一个信息产品（PDF/模板），测试反应
- 咨询：开始接1:1咨询，积累案例

**KPI检查点：**
- 第9个月：开始第一单咨询
- 第12个月：MRR ≥ ¥10,000 + $800

### Phase 3（第13-18个月）：工具/SaaS探索

**前提：** Phase 2稳定后

**探索方向：**
- 根据订阅者反馈，确定工具/SaaS方向
- 最小可行产品（MVP）测试

### Phase 4（第18个月+）：规模化 + B端拓展

**订阅规模：** 追求MRR ¥30,000 + $3,000
**咨询/内训：** 稳定每周2-3单咨询
**工具/SaaS：** 正式产品化

## 七、关键原则

1. **订阅优先，其他让路**：在订阅MRR未达到¥5,000+$500之前，不启动其他变现模式
2. **内容质量 > 数量**：宁可少发，不要发水文
3. **信任是基础**：付费产品必须超预期交付
4. **数据驱动**：每月复盘KPI，不行就调整
5. **长期主义**：订阅是慢活，但也是最稳的收入来源

## 八、风险与规避

| 风险 | 规避方法 |
|-----|---------|
| 内容枯竭 | 建立素材库（行业观察、读书笔记、Maker项目复盘） |
| 平台依赖 | 多平台运营，私域（微信）沉淀 |
| 订阅增长慢 | 优化转化路径，测试不同免费钩子 |
| 时间瓶颈 | 聚焦最有效的平台（公众号+X vs 全平台） |
